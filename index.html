HTML

  • Elements that are nested within other elements are called child elements. The element that holds all the children is known as the parent element
  • The head element contains metadata that helps the browser to render the page correctly
  • The body element contains the content that is shown to the user
  • Header 1 through 6 represent the size given to the text. 1 being largest, 6 being smallest. It's best practice to only use heading 1 a single time per page for user readibility
  • the 'ul' tag is used for unordered lists, 'ol' for ordered lists, and 'li' is used for list items within the previous two tags
  • Image elements used within the 'image' tag are 'src=""', used to specify the location of the image file, and 'alt=""', which is the description of the appearance or functionality of said image

CSS

  • External CSS style sheets are a common way to style a page and are preferred in most cases because you can keep all your CSS rules in a separate file, making it easier to design
  • CSS box model
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • When '*' is used as a selector it applies to ALL elements on the page
  • You can combine two selectors into a single CSS rule with a comma so the declarations affect both selectors
  • Class attributes allow us to share a CSS rule to any element by assigning the rule to said class attribute with a class selector. On this pages CSS the period in '.card' designates it as a class selector
  • You can set multiple effects separated by commas to reduce repetitive and inefficient code

Git

  • Git is essentially navigating your files without the use of a GUI
  • 'ls': used to list all items in current folder
  • 'md 'FOLDERNAME'': is used to change active folder
  • 'cd ..': moves back a folder in directory
  • 'mkdir 'FOLDERNAME'': creates a new folder within current directory
  • 'touch 'FILENAME.txt'': creates an empty .txt file with designated name in current directory
  • 'start 'FILENAME'': will open the file specified
  • 'code .' will open VSC
  • 'git status': checks what branch you are currently on
  • 'git checkout -b branch-name': creats a new branch and switches to it
  • 'git add -A' will commit all files in the working branch

JavaScript

  • To declare a variable use the var keywoard, after that a name, then an assignment operator (=) to assign the specific data stored in the variable. "var topics = "HTML, CSS, Git, JavaScript";" is 4 items stored under 1 variable
  • 'console.log()' will output the specified message to the browser console
  • 'Control Flow' is the order in which a computer executes code in a script. JS runs from the first line to the last. ie you cannot make use of variables that haven't been specified
  • You can interrupt code flow using conditional statements and loops to control the order in which code is executed, if at all
  • A value is considered 'falsy' if it is one of the following values: 0, -0, On, "", null, undefined, nan, or a boolean data type with the value of 'false'
  • All other values, including a boolean with the value of 'true' are considered 'truthy'
  • 'Truthy' and 'Falsy' are adorable ways to consider data
  • An array is a single variable that is used to hold a group of data. ie: var shapes = ["triangle", "square", "pentagon", "circle"]; the variable 'shapes' accounts for all 4 individual data points
  • Numbers and boolean values (true/false) can be used as-is within an array, but strings (names, words, etc.) must be within quotation marks
  • Items within an array are numbered starting at 0 and continue sequentially. So an array with 4 items would begin with 0 for the first item and 3 for the final item
  • for loops use 3 statements: Declaring a variable and giving it a value, the condition, the statement that allows the for statement to iterate over each item
  • x++ is effectively shorthand for x+1
  • ''.length' is a modifier that counds how many items are in an array. 'shapes.length' would be the length of the shapes array numerically
  • A function is a set of instructions that tells the computer how to perform a certain task. It does not execute automatically when the JavaScript file is run, you have to call the function by name to execute it
