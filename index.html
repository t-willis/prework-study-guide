Prework Study Guide

HTML

  • Elements that are nested within other elements are called child elements. The element that holds all the children is known as the parent element
  • The head element contains metadata that helps the browser to render the page correctly
  • The body element contains the content that is shown to the user
  • Header 1 through 6 represent the size given to the text. 1 being largest, 6 being smallest. It's best practice to only use heading 1 a single time per page for user readibility
  • the 'ul' tag is used for unordered lists, 'ol' for ordered lists, and 'li' is used for list items within the previous two tags
  • Image elements used within the 'image' tag are 'src=""', used to specify the location of the image file, and 'alt=""', which is the description of the appearance or functionality of said image

CSS

  • External CSS style sheets are a common way to style a page and are preferred in most cases because you can keep all your CSS rules in a separate file, making it easier to design
  • CSS box model
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • When '*' is used as a selector it applies to ALL elements on the page
  • You can combine two selectors into a single CSS rule with a comma so the declarations affect both selectors
  • Class attributes allow us to share a CSS rule to any element by assigning the rule to said class attribute with a class selector. On this pages CSS the period in '.card' designates it as a class selector
  • You can set multiple effects separated by commas to reduce repetitive and inefficient code

Git

  • Git is essentially navigating your files without the use of a GUI
  • 'ls': used to list all items in current folder
  • 'md 'FOLDERNAME'': is used to change active folder
  • 'cd ..': moves back a folder in directory
  • 'mkdir 'FOLDERNAME'': creates a new folder within current directory
  • 'touch 'FILENAME.txt'': creates an empty .txt file with designated name in current directory
  • 'start 'FILENAME'': will open the file specified
  • 'code .' will open VSC
  • 'git status': checks what branch you are currently on
  • 'git checkout -b branch-name': creats a new branch and switches to it
  • 'git add -A' will commit all files in the working branch
